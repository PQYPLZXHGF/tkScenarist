'Project' menu

'New' menu option

This menu option fully resets application software in order to work with a new virgin project.

A pending modified project will show confirmation dialog before going on with this menu option.

Return to summary.

'Open' menu option

This menu option shows up a file dialog to help user in opening a previously saved project file.

A pending modified project will show confirmation dialog before going on with this menu option.

Return to summary.

'Save' menu option

This menu option actually saves current project data into a file location on hard drive disk.

Never saved projects will first go to 'Save as...' option.

Return to summary.

'Save as...' menu option

This menu option pops up a 'Save as...' file dialog to help user in choosing a new file location/name for current project's data.

Return to summary.

'Export to PDF' menu option

This menu option shows up a dedicated multi-document exportation dialog tool.

Application software will refuse showing up this dialog until project has been saved at least once.

Return to summary.

'Quit' menu option

This menu option simply quits running application.

A pending modified project will show confirmation dialog before going on with this menu option.

Return to summary.

'Edit' menu

'Undo' menu option

This menu option cancels last text entry on a text editor.

Form entries (title, subtitle, etc) do NOT apply.

Return to summary.

'Redo' menu option

This menu option redoes last cancelled text entry on a text editor.

Form entries (title, subtitle, etc) do NOT apply.

Return to summary.

'Select all' menu option

This menu option selects all text onto text editors AND form entries.

Return to summary.

'View' menu

'Toggle status bar' menu option

This menu option toggles display of the status bar (information line at the bottom end of the app window).

If checked, the option displays status bar.

If unchecked, the option hides status bar.

Return to summary.

'Help' menu

'Offline documentation' menu option

This menu option launches the default internet browser and displays local documentation embedded with the software package, in HTML format.

This option allows documenting even when there is no more internet access.

Anyway, one may prefer to read out online documentation whenever possible, as official documentation is mostly better up-to-date than offline documentation.

Return to summary.

'Online documentation' menu option

This menu option launches the default internet browser and displays official online documentation.
