Contributeurs

Contribuer au projet

tkScenarist a toujours besoin de personnes créatives pour :

  • écrire de nouveaux modèles d'histoires / de pitchs tels que 'love story', 'science-fiction', 'thriller', 'fantastique', 'super-héros', 'contes et légendes', etc – tout ce que vous aimeriez partager avec le plus grand nombre (voir 'Modèles d'histoires/pitchs') ;
  • traduire le logiciel dans votre langue maternelle (italien, espagnol, portugais, allemand, hollandais, polonais, chinois, etc) ;
  • traduire la documentation wiki ici présente dans votre langue maternelle (italien, espagnol, portugais, allemand, hollandais, polonais, chinois, etc) : veuillez contacter l'équipe admin pour devenir contributeur officiel et commencer vos traductions dès que possible ;
  • béta-testeurs : testez la dernière version en date du logiciel et signalez sur le bugtracker de GitHub tous les problèmes que vous rencontrez.

Les personnes désireuses d'apporter des améliorations aux fichiers du projet sont invitées à prendre connaissance de la documentation officielle GitHub concernant les forks ainsi que les pull requests.

Merci pour votre aide précieuse !
